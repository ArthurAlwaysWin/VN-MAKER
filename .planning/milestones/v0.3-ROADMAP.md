## Overview

将游戏内容编辑器从命令时间线模式升级为 PPT 页面模式，让用户像编辑幻灯片一样创建游戏内容。包含页面数据架构、所见即所得编辑器、资源选择器、转场与分支系统、以及内联试玩预览。

## Key Accomplishments

1. **Page-based data schema** — Self-contained pages with background/characters/dialogue/BGM/SE, 3 page types (normal/choice/condition), engine fully rewritten for page-based playback
2. **PPT page editor** — Left sidebar with scene tree, 1280×720 WYSIWYG canvas with draggable characters, inspector panel for dialogue/BGM/SE editing
3. **Visual resource pickers** — Character expression thumbnail grid, background previews, audio picker with inline playback
4. **Transitions & branching** — Fade/slide transitions per page, choice pages with option-to-scene linking, setVariable support, visual distinction (🔀 badge)
5. **Inline test play** — iframe-based preview with postMessage protocol, play/stop/mute controls, read-only overlay, asset:// path resolution
6. **UI polish** — Speaker combobox, choice options canvas preview, character scale slider with draggable handle
