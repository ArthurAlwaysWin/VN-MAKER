**Fix 1: DialogueBox.js speaker name color (line 72)**

Current code:
```javascript
this.nameEl.style.color = data.speakerColor || this._activeNameplateColor || '#fff';
```

Change to:
```javascript
const nameColor = data.speakerColor || this._activeNameplateColor;
if (nameColor) {
  this.nameEl.style.color = nameColor;
}
// else: CSS var(--gm-text, #fff) handles the default
```

This preserves the D-04 priority chain:
- Per-character data.speakerColor (highest) sets inline, overrides everything
- Global nameplate this._activeNameplateColor from ui.dialogueBox.nameplateColor sets inline, overrides CSS
- No explicit color: NO inline style set, CSS var(--gm-text, #fff) cascade applies (theme token level)
- No theme token set: CSS fallback #fff applies (v0.5 default)

Do NOT change _applyFontSettings() (lines 148-173). It only sets inline styles when user-specified values exist (all guarded by if checks). These are content layer per D-01, correctly overriding tokens.

Do NOT change _applyStyle() per-dialogue overrides (lines 103-128). These are per-dialogue custom values from script.json, correctly overriding tokens per D-04.

**Fix 3: BacklogScreen.js speaker color (lines 39-46)**

Current code:
```javascript
const speakerColor = entry.speaker && characters[entry.speaker]
  ? characters[entry.speaker].color
  : 'rgba(255,255,255,0.5)';

div.innerHTML = entry.speakerName
  ? `<div class="backlog-speaker" style="color:${speakerColor}">${entry.speakerName}</div>
     <div class="backlog-text">${entry.text}</div>`
  : `<div class="backlog-text" style="font-style:italic">${entry.text}</div>`;
```

Change to conditionally set inline color only when character has a defined color:
```javascript
const charColor = entry.speaker && characters[entry.speaker]
  ? characters[entry.speaker].color
  : null;
const speakerStyle = charColor ? ` style="color:${charColor}"` : '';

div.innerHTML = entry.speakerName
  ? `<div class="backlog-speaker"${speakerStyle}>${entry.speakerName}</div>
     <div class="backlog-text">${entry.text}</div>`
  : `<div class="backlog-text" style="font-style:italic">${entry.text}</div>`;
```

Priority chain:
- Character with defined color: inline style set, overrides CSS token
- Character without color or narrator: NO inline style, CSS var(--gm-text-muted, rgba(255,255,255,0.5)) cascade applies
- The font-style:italic on narrator text stays (structural, not a color)
