Create `src/editor/components/ExportModal.vue` — a single-file Vue 3 component with `<script setup>`, scoped CSS, dark theme.

**Props and Emits:**
```javascript
const props = defineProps({
  visible: Boolean,
});
const emit = defineEmits(['close']);
```

**State machine — 3 reactive refs:**
```javascript
const state = ref('config'); // 'config' | 'exporting' | 'done'
const progress = ref({ step: '', percent: 0 });
const result = ref(null); // { success, outputPath, zipPath, warnings } or { success: false, error }
```

**Config form fields:**
```javascript
const gameTitle = ref('');
const outputDir = ref('');
const faviconPath = ref(null);
const enableZip = ref(false);
const warningsExpanded = ref(false);
```

**On visible change (watch):** When `visible` becomes true, reset all state: `state.value = 'config'`, `gameTitle.value = project.projectData?.name || ''`, `outputDir.value = ''`, `faviconPath.value = null`, `enableZip.value = false`, `result.value = null`, `warningsExpanded.value = false`.

**Methods:**

1. `async pickOutputDir()` — Calls `window.ipcRenderer.invoke('dialog-open-directory')`. If result is non-null string, sets `outputDir.value = result`.

2. `async pickFavicon()` — Calls `window.ipcRenderer.invoke('dialog-open-file', { title: '选择 Favicon', filters: [{ name: '图标文件', extensions: ['ico', 'png'] }] })`. If result is non-null, sets `faviconPath.value = result`.

3. `clearFavicon()` — Sets `faviconPath.value = null`.

4. `async startExport()` — The main export flow:
   - Validate: if `!gameTitle.value.trim()` or `!outputDir.value`, return early (don't start)
   - Set `state.value = 'exporting'`, `progress.value = { step: '准备中...', percent: 0 }`
   - Register progress listener: `const unsub = window.ipcRenderer.on('export-progress', (event, payload) => { progress.value = payload; })`
   - Call: `const res = await window.ipcRenderer.invoke('export-game', { outputDir: outputDir.value, gameTitle: gameTitle.value.trim(), faviconPath: faviconPath.value, zip: enableZip.value })`
   - Unsubscribe: `unsub()`
   - If `state.value !== 'exporting'` return (user cancelled while waiting)
   - Set `result.value = res`, `state.value = 'done'`

5. `cancelExport()` — Sets `state.value = 'config'` (returns to config form; background export may complete but result is ignored per D-05).

6. `async openOutputFolder()` — Calls `window.ipcRenderer.invoke('open-folder', result.value.outputPath)` per D-09.

7. `retry()` — Sets `state.value = 'config'` per D-11.

8. `onClose()` — If `state.value === 'exporting'`, do nothing (prevent close during export). Otherwise, `emit('close')`.

        <template v-if="state === 'config'">
