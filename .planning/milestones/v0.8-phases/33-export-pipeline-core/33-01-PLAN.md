**Step 3 — Create default game icon (D-02):**

After npm install completes, `pngjs` is available as a transitive dependency of `png-to-ico`. Run a Node.js script to generate a valid 256×256 PNG:

```bash
node --input-type=module -e "
import { PNG } from 'pngjs';
import { writeFileSync } from 'node:fs';
const size = 256;
const img = new PNG({ width: size, height: size });
for (let y = 0; y < size; y++) {
  for (let x = 0; x < size; x++) {
    const idx = (size * y + x) << 2;
    img.data[idx]     = 0x2d;  // R (dark purple #2d1b69)
    img.data[idx + 1] = 0x1b;  // G
    img.data[idx + 2] = 0x69;  // B
    img.data[idx + 3] = 0xff;  // A
  }
}
writeFileSync('public/default-game-icon.png', PNG.sync.write(img));
console.log('Created default-game-icon.png');
"
```

This creates a solid dark purple (matching the project's dark theme) 256×256 PNG. The file must be a valid PNG that `png-to-ico` can later convert to .ico.
