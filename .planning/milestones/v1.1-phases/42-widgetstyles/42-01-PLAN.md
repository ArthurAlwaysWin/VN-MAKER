    ```
    /**
     * PanelWidget — applies panel styling from widgetStyles.panel config.
     * Supports background color, background image with opacity, border,
     * borderRadius, backdropBlur, padding, and nineSlice backgrounds.
     */
    ```

    Export function `applyPanelStyle(el, config)`:
    - `el` is an existing DOM element to style
    - `config` is the merged widgetStyles.panel object (from deepMergeWidgetStyles result)
    - Apply inline styles:
      - `el.style.background = sanitizeCssValue(config.background) || ''`
      - `el.style.borderRadius = (typeof config.borderRadius === 'number') ? config.borderRadius + 'px' : ''`
      - `el.style.border = sanitizeCssValue(config.border) || ''`
      - `el.style.backdropFilter = (typeof config.backdropBlur === 'number') ? 'blur(' + config.backdropBlur + 'px)' : ''`
      - Also set `-webkit-backdrop-filter` for Safari compat
      - `el.style.padding` = if config.padding is array `[top, side]` → `${top}px ${side}px`, if number → `${n}px`, else ''
    - If `config.backgroundImage` is truthy and passes sanitizeCssValue:
      - Create a child div with class `gm-panel-bg-layer`
      - Position absolute, inset 0, z-index -1
      - background-image: `url("${resolvePath(config.backgroundImage)}")`
      - background-size: cover, background-position: center
      - opacity: `config.backgroundImageOpacity ?? 0.3`
      - pointer-events: none
      - Set el.style.position to 'relative' if not already set, el.style.isolation = 'isolate'
      - Append child to el
    - If `config.nineSlice` is truthy and `config.nineSlice.src` exists:
      - Create a child div with class `gm-panel-nine`
      - Position absolute, inset 0, z-index -1, pointer-events: none
      - Build border-image from nineSlice config following ThemeManager pattern:
        - slice: `(config.nineSlice.slice || [0,0,0,0]).join(' ') + ' fill'`
        - width: `(config.nineSlice.width || config.nineSlice.slice || [0,0,0,0]).map(v => v + 'px').join(' ')`
        - outset: `(config.nineSlice.outset || [0,0,0,0]).map(v => v + 'px').join(' ')`
        - repeat: `config.nineSlice.repeat || 'stretch'`
      - Apply: `border-image: url("${resolvePath(config.nineSlice.src)}") ${slice} / ${width} / ${outset} ${repeat}`
      - Set el position/isolation same as backgroundImage case
      - Append to el
    - Import `sanitizeCssValue` from `'../sanitize.js'` and `resolvePath` from `'../../engine/assetPath.js'`
