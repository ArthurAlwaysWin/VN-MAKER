# Plan 45-02: main.js Config Routing + Editor Preview

## Goal
Add centralized `ui.*` config application in `init()` and `initPreview()` so all UI
components receive their layout/style configs after engine.load().

## Context
- main.js: ~920 lines. init() at line 782, initPreview() at line 845.
- Already has:
  - Line 804-806: `if (ui.titleScreen) titleScreen.setLayout(ui.titleScreen)`
  - Line 809-811: `if (ui.settingsScreen) settingsScreen.setLayout(ui.settingsScreen)`
  - Line 818-820: `if (ui.dialogueBox) dialogueBox.applyGlobalStyle(ui.dialogueBox)`
  - Line 864-866: theme + nineSlice in preview
  - Line 869-871: dialogueBox.applyGlobalStyle in preview
- Need to ADD:
  - saveLoadScreen.setLayout(ui.saveLoadScreen)
  - backlogScreen.setLayout(ui.backlogScreen)
  - gameMenu.setLayout(ui.gameMenu)
  - settingsScreen.setWidgetStyles(ui.widgetStyles)
  - dialogueBox.setNameplateStyle(ui.dialogueBox.nameplateStyle) — in both init() and preview

## Tasks

### Task 1: Add config routing in init()
**File:** `src/main.js`
**Action:** After existing settingsScreen.setLayout block (line 811), add:

```js
// Apply widget styles for settings controls (Phase 42)
if (engine.script.ui?.widgetStyles) {
  settingsScreen.setWidgetStyles(engine.script.ui.widgetStyles);
}

// Apply screen layouts (Phase 43)
if (engine.script.ui?.saveLoadScreen) {
  saveLoadScreen.setLayout(engine.script.ui.saveLoadScreen);
}
if (engine.script.ui?.backlogScreen) {
  backlogScreen.setLayout(engine.script.ui.backlogScreen);
}
if (engine.script.ui?.gameMenu) {
  gameMenu.setLayout(engine.script.ui.gameMenu);
}

// Apply nameplate style (Phase 45)
if (engine.script.ui?.dialogueBox?.nameplateStyle) {
  dialogueBox.setNameplateStyle(engine.script.ui.dialogueBox.nameplateStyle);
}
```

### Task 2: Add config routing in initPreview()
**File:** `src/main.js`
**Action:** In the 'start' message handler, after existing dialogueBox.applyGlobalStyle block (line 871), add the same config blocks:

```js
// Apply widget styles for preview
if (engine.script.ui?.widgetStyles) {
  settingsScreen.setWidgetStyles(engine.script.ui.widgetStyles);
}

// Apply screen layouts for preview
if (engine.script.ui?.settingsScreen) {

  saveLoadScreen.setLayout(engine.script.ui.saveLoadScreen);
```

Note: settingsScreen.setLayout already exists in init() but NOT in initPreview(). Add it here.
Note: titleScreen.setLayout is not needed in preview (preview starts directly at a scene).
