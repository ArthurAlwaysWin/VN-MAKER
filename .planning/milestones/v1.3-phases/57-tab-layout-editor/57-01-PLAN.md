**File 1: `src/editor/components/layout/tabLayoutHelpers.js`**

Create a pure JS module with JSDoc header: "Tab & Layout Editor helpers — pure data functions for tab CRUD and setting assignment. Testable without Vue/JSDOM."

Export these named functions and constants:

1. `DEFAULT_TABS` — hardcoded array matching engine defaults (engine doesn't export these):
```javascript
export const DEFAULT_TABS = [
  { label: '声音', settingKeys: ['master-volume', 'bgm-volume', 'se-volume', 'voice-volume'] },
  { label: '画面', settingKeys: ['dialogue-opacity', 'window-mode'] },
  { label: '游戏', settingKeys: ['text-speed', 'auto-speed', 'skip-mode'] },
];
```

2. `ensureDefaultTabs(cfg)` — If `cfg.tabBar.tabs` is absent or empty array, deep-copy `DEFAULT_TABS` into it. Returns `cfg` (or null if cfg is falsy). Must use `cfg.tabBar ??= {};` and `map(t => ({ ...t, settingKeys: [...t.settingKeys] }))` to avoid mutating the constant.

3. `addTab(cfg)` — Ensures `cfg.tabBar.tabs` exists (??= []), pushes `{ label: '新标签', settingKeys: [] }`. Per D-04.

4. `deleteTab(cfg, index)` — Splices `cfg.tabBar.tabs` at index. Guard: return early if `!cfg?.tabBar?.tabs`. Per D-05.

5. `setTabLabel(cfg, index, label)` — Sets `cfg.tabBar.tabs[index].label = label`. Guard: return early if tab doesn't exist. Per D-06.

6. `setTabIcon(cfg, index, icon)` — Sets `cfg.tabBar.tabs[index].icon = icon`. Per D-08.

7. `toggleKeyAssignment(cfg, key, tabIndex)` — Per D-12 exclusive assignment:
   - Record `wasInTarget = tabs[tabIndex]?.settingKeys?.includes(key)`
   - Remove `key` from ALL tabs' settingKeys arrays (iterate, indexOf, splice)
   - If `!wasInTarget` AND `tabs[tabIndex]` exists: push `key` to `tabs[tabIndex].settingKeys` (ensure array with ??= [])
   - If `wasInTarget`: do nothing after removal (key is now unassigned = toggle off)

8. `getUnassignedKeys(tabs, allKeys)` — Collect all assigned keys into a Set, return `allKeys.filter(k => !assigned.has(k))`. Per D-14.

9. `isKeyInTab(tabs, key, tabIndex)` — Returns `tabs?.[tabIndex]?.settingKeys?.includes(key) ?? false`.

All functions use `export function` (named exports, no default export — per project conventions).
