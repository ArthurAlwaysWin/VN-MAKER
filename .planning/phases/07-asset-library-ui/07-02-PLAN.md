**Create src/editor/components/resource-library/AssetGrid.vue — Backgrounds thumbnail grid:**

Props: `category` (String, default 'backgrounds')
Uses: `useAssetStore`, InlineEdit, ContextMenu

Template:
```html
<!-- Empty state when no files -->
<div v-if="fileList.length === 0" class="empty-state">
  <div class="empty-icon">📁</div>
  <p class="empty-title">当前分类下暂无背景图</p>
  <p class="empty-subtitle">拖放文件到此处，或点击上方"导入文件"按钮</p>
</div>

<!-- Thumbnail grid -->
<div v-else class="asset-grid">
  <div class="asset-card" v-for="file in fileList" :key="file" @contextmenu.prevent="showMenu($event, file)">
    <div class="thumbnail">
      <img :src="`asset://${category}/${file}`" :alt="file" />
    </div>
    <div class="filename">
      <InlineEdit :ref="el => setEditRef(file, el)" :value="file" :preserveExtension="true" @save="onRename(file, $event)" />
    </div>
  </div>
</div>

<ContextMenu :visible="menuVisible" :x="menuX" :y="menuY" :items="menuItems" @select="onMenuAction" @close="menuVisible = false" />
```

Script:
- `fileList` computed: `assets.files[props.category]` (this is a reactive array of filename strings)
- `menuVisible`, `menuX`, `menuY` refs. `menuTarget` ref (current filename for menu actions).
- `menuItems`: `[{ label: '重命名', action: 'rename' }, { separator: true }, { label: '删除', action: 'delete', destructive: true }]`
- `showMenu(event, file)`: set menu position and target, show menu
- `onMenuAction(action)`: if 'rename', call the InlineEdit's `startEdit()` on the targeted file's ref. If 'delete', call `onDelete(menuTarget)`.
- `onDelete(filename)`: `if (confirm('确定要删除 "' + filename + '" 吗？')) { await assets.deleteAsset(category, filename); }`
- `onRename(oldName, newName)`: if `oldName !== newName`, call `await assets.renameAsset(category, oldName, newName)`. If `!result.success`, `alert(result.error)`.
- `editRefs` object to track InlineEdit refs per filename (use `setEditRef(file, el)` pattern with a plain object).
