## Page Types

### Normal Page (D-01, D-03)
```javascript
{
  id: 'string',                // Unique within scene, e.g. 'p1', 'p2'
  type: 'normal',
  background: 'string|null',   // Asset path e.g. 'backgrounds/school_gate.svg'
  characters: [                // D-03: complete declaration, no inheritance
    {
      id: 'string',            // Character ID from top-level characters{}
      expression: 'string',    // Expression name for this page's initial state
      position: 'string',      // 'left'|'center'|'right'|'custom'
      x: number|null,          // Custom x (only when position='custom')
      y: number|null,
      scale: number            // Default 1
    }
  ],
  bgm: {                       // D-03: self-contained, null = stop BGM
    file: 'string',
    volume: number             // 0-1
  } | null,
  se: {                        // Sound effect played when page renders
    file: 'string'             // Asset path e.g. 'audio/se_footstep.mp3'
  } | null,
  dialogues: [                 // D-01: multiple dialogues per page
    {
      speaker: 'string|null',  // Character ID or null for narration
      text: 'string',
      expression: 'string|null' // Speaker's expression change, null = no change
    }
  ],
  transition: {
    type: 'string',            // 'fade'|'none' (more in Phase 13)
    duration: number           // milliseconds
  }
}
```

### Choice Page (D-04)
```javascript
{
  id: 'string',
  type: 'choice',
  background: 'string|null',
  characters: [...],           // Same as normal page
  bgm: {...} | null,
  prompt: 'string',
  options: [
    {
      text: 'string',
      target: 'string',       // Target scene ID to jump to
      setVariable: object|undefined  // Optional { varName: deltaValue }
    }
  ],
  dialogues: [],               // Always empty for choice pages
  transition: {...}
}
```

### Condition Page
```javascript
{
  id: 'string',
  type: 'condition',
  variable: 'string',         // Variable name to check
  operator: 'string',         // '=='|'!='|'>'|'>='|'<'|'<='
  value: number,
  trueTarget: 'string',       // Scene ID if condition true
  falseTarget: 'string'       // Scene ID if condition false
}
```

## Scene Structure (D-02)
```javascript
{
  name: 'string',
  next: 'string|undefined',   // Optional: scene ID to jump to after last page
  pages: [...]                 // Array of page objects
}
```

## Conversion Rule (D-01)
New page is created when: background changes, or character is added/removed.
Dialogues within the same visual state go into the same page's dialogues[] array.
Expression changes during dialogue use the dialogue entry's `expression` field.
