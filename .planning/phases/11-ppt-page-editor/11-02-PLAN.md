This is a `<script setup>` SFC implementing the full left sidebar per UI-SPEC "Scene Tree" specification and decisions D-01 through D-04.

**Template structure** (see Research Pattern 5 for the nested v-for approach — do NOT use recursive components, it's only 2 levels deep):

```html
<div class="scene-tree">
  <!-- Tree content (scrollable) -->
  <div class="tree-content">
    <!-- Empty state when no scenes -->
    <div v-if="sceneEntries.length === 0" class="empty-state">
      暂无场景，点击下方按钮创建第一个场景
    </div>

    <!-- Scene nodes -->
    <div v-for="[sceneId, scene] in sceneEntries" :key="sceneId" class="scene-node">
      <!-- Scene header row -->
      <div class="scene-header" @click="toggleExpand(sceneId)" @dblclick="startRenameScene(sceneId, scene.name)">
        <span class="expand-icon">{{ expanded[sceneId] ? '▼' : '▶' }}</span>
        <span class="scene-icon">🎬</span>
        <span class="scene-name">{{ scene.name }}</span>
        <button class="scene-menu-btn" @click.stop="showSceneMenu(sceneId, $event)" title="场景操作">⋯</button>
      </div>

      <!-- Page items (when expanded) -->
      <div v-if="expanded[sceneId]" class="scene-pages">
        <!-- Empty pages state -->
        <div v-if="!scene.pages || scene.pages.length === 0" class="empty-pages">
          当前场景暂无页面，点击「+ 添加页面」开始创建
        </div>

        <!-- Page items with drag support -->
        <div v-for="(page, idx) in scene.pages" :key="page.id"
          class="page-item"
          :class="{ active: isSelected(sceneId, idx), 'drag-over': dragTarget.sceneId === sceneId && dragTarget.index === idx }"
          draggable="true"
          @click="onSelectPage(sceneId, idx)"
          @dragstart="onDragStart($event, sceneId, idx)"
          @dragover="onDragOver($event, sceneId, idx)"
          @dragleave="onDragLeave"
          @drop="onDrop($event, sceneId, idx)"
          @dragend="onDragEnd">
          <span class="page-icon">{{ pageIcon(page.type) }}</span>
          <span class="page-number">{{ idx + 1 }}</span>
          <span class="page-snippet">{{ pageSnippet(page) }}</span>
        </div>
      </div>
    </div>
  </div>

  <!-- Footer actions (fixed at bottom) -->
  <div class="tree-footer">
    <button class="footer-btn" @click="onAddPage">+ 添加页面</button>
    <button class="footer-btn" @click="onAddScene">+ 添加场景</button>
  </div>

  <!-- Context menu (simple positioned popup) -->
  <div v-if="contextMenu.visible" class="context-menu" :style="contextMenuStyle" @click.stop>
    <div class="menu-item" @click="onRenameScene">重命名</div>
    <div class="menu-item" @click="onAddPageToScene">添加页面</div>
    <div class="menu-item danger" @click="onDeleteScene">删除</div>
  </div>
</div>
```

**Script setup logic:**

1. **Imports:** `ref, computed, reactive, onMounted, onBeforeUnmount` from vue; `usePageEditor` from composable; `useScriptStore` from store.

2. **State injection:**
   ```javascript
   const { script, selectedSceneId, selectedPageIndex, selectPage } = usePageEditor();
   ```

3. **Local state:**
   - `expanded = reactive({})` — tracks which scenes are expanded (auto-expand first scene on mount)
   - `dragState = ref({ sceneId: null, fromIndex: -1 })` — current drag source
   - `dragTarget = reactive({ sceneId: null, index: -1 })` — current drop target (for visual indicator)
   - `contextMenu = reactive({ visible: false, sceneId: null, x: 0, y: 0 })` — right-click menu

4. **Computed:**
   - `sceneEntries = computed(() => Object.entries(script.data?.scenes || {}))` — scene list for v-for

5. **Helper functions:**
   - `pageIcon(type)` — returns `'📑'` for normal, `'🔀'` for choice, `'❓'` for condition (per D-04)
   - `pageSnippet(page)` — returns first dialogue text truncated to ~20 chars, or `'[选择页]'` for choice, `'[条件页]'` for condition (per D-04). For normal pages with no dialogues, return empty string.
   - `isSelected(sceneId, idx)` — returns `selectedSceneId.value === sceneId && selectedPageIndex.value === idx`
   - `toggleExpand(sceneId)` — toggle `expanded[sceneId]`

6. **Page selection:**
   - `onSelectPage(sceneId, idx)` — call `selectPage(sceneId, idx)`. Auto-expand the scene if collapsed.

7. **Scene management (per D-02):**
   - `onAddScene()` — Use `prompt('输入新的场景名称')` for scene name. Generate sceneId: sanitize name to snake_case or use `'scene_' + Date.now()`. Call `script.addScene(sceneId, sceneName)`. Auto-expand the new scene and select its first page.
   - `startRenameScene(sceneId, currentName)` — Use `prompt('输入新的场景名称', currentName)`. If user provides new name, call `script.renameScene(sceneId, newName)`.
   - `onRenameScene()` — calls `startRenameScene(contextMenu.sceneId, ...)`, closes context menu.
   - `onDeleteScene()` — Use `confirm('确定删除场景「' + sceneName + '」及其所有页面？')`. If confirmed, call `script.deleteScene(sceneId)`. If deleted scene was selected, select the first remaining scene. Close context menu.
   - `showSceneMenu(sceneId, event)` — set contextMenu position to `event.clientX/Y`, set sceneId, show menu.

8. **Page management (per D-03):**
   - `onAddPage()` — Call `script.addPage(selectedSceneId.value, selectedPageIndex.value)`. If result, select the new page at result.index.
   - `onAddPageToScene()` — Add page to the context menu's sceneId at the end (afterIndex = -1). Close context menu.
   - Deleting pages: on page-item right-click or add a small ✕ button. For simplicity, implement via keyboard: listen for Delete key on selected page. Use `confirm('确定删除第 ' + (selectedPageIndex.value + 1) + ' 页？此操作不可撤销。')`. If confirmed, call `script.deletePage(sceneId, pageIndex)`. Adjust selection to previous page or 0.

9. **Drag-reorder (per EDITOR-04, Research Example 2):**
   - `onDragStart(e, sceneId, idx)` — Set `dragState`, `e.dataTransfer.effectAllowed = 'move'`, `e.dataTransfer.setData('text/plain', String(idx))`. Set dragged element opacity via `e.target.style.opacity = '0.5'`.
   - `onDragOver(e, sceneId, idx)` — Only allow if same scene as dragState. `e.preventDefault()`, `e.dataTransfer.dropEffect = 'move'`. Set `dragTarget` for visual drop indicator.
   - `onDragLeave()` — Reset `dragTarget`.
   - `onDrop(e, sceneId, toIndex)` — `e.preventDefault()`. Call `script.reorderPages(sceneId, dragState.fromIndex, adjustedToIndex)`. Select the moved page at its new position. Reset dragState.
   - `onDragEnd()` — Reset opacity and dragState.
   - **Adjust toIndex:** If `fromIndex < toIndex`, subtract 1 from toIndex (because the element was already removed from its original position before insert).

10. **Global click listener** to close context menu: `onMounted` → `document.addEventListener('click', closeMenu)`, clean up in `onBeforeUnmount`.

11. **Delete key handler:** `onMounted` → listen for keydown Delete on document. If a page is selected, show confirm and delete.
