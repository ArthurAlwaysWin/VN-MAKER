```javascript
const assetStore = useAssetStore();
```

**Watch:** When `visible` becomes true, reset state and sync tab:

```javascript
watch(() => props.visible, (val) => {

    selectedFile.value = null;
    activePlayerId.value = null;
  }
});
```

- `switchTab(tab)` — sets `activeTab.value = tab`, resets `selectedFile.value = null`, pauses audio (`activePlayerId.value = null`)
- `onOverlayClick(e)` — if `e.target === e.currentTarget`, emit `'close'`
