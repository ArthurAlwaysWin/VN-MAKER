Create `src/engine/ReadHistory.js` per D-12 — a standalone class that tracks which pages have been read.

**Class: ReadHistory**

Constructor:
- `constructor(projectId)` — stores `this._storageKey = \`readHistory:${projectId}\``; creates `this._read = new Set()`; calls `this._load()` (per D-03: localStorage with key format `readHistory:{projectId}`)

Methods:
- `markRead(sceneId, pageIndex)` — creates key string `\`${sceneId}:${pageIndex}\``, returns early if already in Set (skip persistence), otherwise adds to Set and calls `this._save()`
- `isRead(sceneId, pageIndex)` — returns `this._read.has(\`${sceneId}:${pageIndex}\`)`
- `clear()` — calls `this._read.clear()` then `this._save()`
- `get size` — returns `this._read.size`

Private methods:
- `_load()` — try/catch wrapping `localStorage.getItem(this._storageKey)`, if raw exists `JSON.parse(raw)` into `new Set(arr)`, catch logs `console.warn('[ReadHistory] Failed to load:', e)`
- `_save()` — try/catch wrapping `localStorage.setItem(this._storageKey, JSON.stringify([...this._read]))`, catch logs `console.warn('[ReadHistory] Failed to save:', e)`

File must have JSDoc comment block at top: `ReadHistory — Tracks which pages have been read by the player`. Use `export class ReadHistory` (named export, no default). Follow underscore prefix for private methods. Use single quotes, 2-space indent, always semicolons.
