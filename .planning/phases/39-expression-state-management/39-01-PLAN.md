<objective>
Implement expression state management in ScriptEngine — a Map tracking each character's current expression name across pages. Provides cross-page inheritance (pages without explicit expression reuse the previous one), fallback to first expression for new characters, scene boundary reset, mid-dialogue expression tracking, and save/load persistence.

Purpose: Enables expression continuity across pages so game creators don't need to re-specify expressions on every page. Completes the STATE requirements for v1.0 expression system.
Output: Modified `src/engine/ScriptEngine.js` with all expression state logic.
