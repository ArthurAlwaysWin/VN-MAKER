# Plan 43-01: SaveLoadScreen.setLayout()

## Goal
Add `setLayout(config)` to SaveLoadScreen that applies background/header/slotGrid/slot/pagination
configuration from `ui.saveLoadScreen` schema. When config is null, existing rendering is unchanged.

## Context
- SaveLoadScreen.js: 295 lines, has _render/_renderGrid/_renderPagination
- Design spec Section 5.1 defines complete JSON schema
- Pattern: SettingsScreen.setLayout() stores config, show() branches on it
- sanitize.js: sanitizeCssValue + clampField for safe CSS
- assetPath.js: resolvePath for backgroundImage paths

## Design Spec Schema (from Section 5.1)
```json
{
  "background": null,
  "backdropBlur": 12,
  "header": {
    "saveTitle": "存 档",
    "loadTitle": "読 档",
    "saveTitleColor": null,
    "loadTitleColor": null,
    "backgroundImage": null,
    "height": 70
  },
  "slotGrid": {
    "columns": 3,
    "rows": 3,
    "gap": 12,
    "x": 60,
    "y": 90,
    "width": 1160,
    "height": 540
  },
  "slot": {
    "background": "rgba(30,30,50,0.6)",
    "backgroundImage": null,
    "borderRadius": 6,
    "border": "1px solid rgba(255,255,255,0.1)",
    "emptyText": "— 空 —",
    "thumbnailRadius": 4
  },
  "pagination": {
    "style": "dots",
    "activeColor": null,
    "inactiveColor": null
  }
}
```

## Tasks

### Task 1: Add setLayout() method and config storage
**File:** `src/ui/SaveLoadScreen.js`
**Action:** Add after constructor:
```js
setLayout(config) {
  this._layoutConfig = config || null;
}
```
Add `this._layoutConfig = null;` in constructor.

### Task 2: Config-driven _render()
**File:** `src/ui/SaveLoadScreen.js`
**Action:** In `_render()`, after building innerHTML, apply config if present:
- `this._layoutConfig.background` → `this.el.style.background`
- `this._layoutConfig.backdropBlur` → `this.el.style.backdropFilter = 'blur(Npx)'`
- `this._layoutConfig.header.saveTitle` / `loadTitle` → title text based on mode
- `this._layoutConfig.header.saveTitleColor` / `loadTitleColor` → title color
- `this._layoutConfig.header.backgroundImage` → header background-image via resolvePath
- `this._layoutConfig.header.height` → header height in px

Pattern: `if (this._layoutConfig) { const cfg = this._layoutConfig; ... }`
If `_layoutConfig` is null, existing hardcoded path runs untouched.

### Task 3: Config-driven _renderGrid()
**File:** `src/ui/SaveLoadScreen.js`
**Action:** After grid.innerHTML = '', if config:
- `cfg.slotGrid.columns` → grid CSS grid-template-columns
- `cfg.slotGrid.rows` → compute SLOTS_PER_PAGE override (columns × rows)
- `cfg.slotGrid.gap` → grid gap
- `cfg.slotGrid.x/y/width/height` → grid positioning

Recalculate `startSlot/endSlot` using dynamic slotsPerPage if config overrides columns/rows.
If null, use existing SLOTS_PER_PAGE = 9.

### Task 4: Config-driven _createSlotCard()
**File:** `src/ui/SaveLoadScreen.js`
**Action:** Apply slot config:
- `cfg.slot.background` → slotEl.style.background
- `cfg.slot.backgroundImage` → slotEl.style.backgroundImage via resolvePath
- `cfg.slot.borderRadius` → slotEl.style.borderRadius
- `cfg.slot.border` → slotEl.style.border
- `cfg.slot.emptyText` → replace '— 空 —' text
- `cfg.slot.thumbnailRadius` → img.style.borderRadius

### Task 5: Config-driven _renderPagination()
**File:** `src/ui/SaveLoadScreen.js`
**Action:** Apply pagination config:
- `cfg.pagination.style === 'dots'` → render dots instead of numbered buttons
- `cfg.pagination.activeColor` → active button/dot color
- `cfg.pagination.inactiveColor` → inactive button/dot color

### Task 6: Dynamic page count calculation
**File:** `src/ui/SaveLoadScreen.js`
**Action:** When config changes columns/rows, total pages = ceil(108 / (columns × rows)).
Add `_getSlotsPerPage()` and `_getTotalPages()` helpers that check config.

### Task 7: Unit tests
**File:** `tests/saveLoadScreenLayout.test.js`
**Action:** Create tests:
1. setLayout(null) → _layoutConfig is null
2. setLayout(config) → _layoutConfig stores config
3. _render with null config → no style changes (COMPAT-02)
4. _render with config.background → el.style.background set
5. _render with config.header.saveTitle → title text changed
6. Pagination with dots style → renders span.page-dot instead of buttons

## COMPAT-02 Verification
The null path must produce identical output:
- `_render()`: if (!this._layoutConfig) → existing innerHTML template unchanged
- `_renderGrid()`: if (!this._layoutConfig) → existing SLOTS_PER_PAGE=9 loop unchanged
- `_createSlotCard()`: if (!this._layoutConfig) → existing slot HTML unchanged
- `_renderPagination()`: if (!this._layoutConfig) → existing numbered buttons unchanged
