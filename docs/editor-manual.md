# Galgame Maker — 可视化编辑器使用手册

欢迎使用 **Galgame Maker** 可视化编辑器！本编辑器专为不具备编程基础的视觉小说创作者设计，您可以通过类似于剪辑视频的时间轴界面，轻松完成角色的增删改查、对话分支的选项配置、以及游戏场景的搭建。

---

## 🚀 1. 快速起步

编辑器依托于本地操作系统运行，支持直接读取您电脑上的素材文件，无需将隐私素材上传到云端。

1. 打开系统终端 (Terminal)，进入您的游戏项目文件夹。
2. 输入 `npm run dev`。
3. 等待几秒后，会自动弹出一个深黑色的独立桌面窗口，这就是您的**编辑器主界面**。

---

## 🎨 2. 界面总览

编辑器的界面灵感来源于专业的代码编辑器（如 VS Code）和游戏引擎（如 Unity）。整体分为三个主要区域：

- **左侧边栏 (Sidebar)**：导航面板。您可以在这里切换“主页(Dashboard)”、“资源管理(Assets)”、“角色配置(Characters)”以及“场景剧本(Scenes)”。左下角常驻 **「▶ Play Preview」** 按钮。
- **中间工作区 (Workspace)**：核心交互区。根据左侧栏选中的不同版块，这里会显示素材网格、角色列表或者指令时间轴。
- **右侧属性检查器 (Inspector)**：参数配置区。当您在中间选中某个具体的角色或场景指令时，右侧会动态显示出可供修改的表单属性。

---

## 📁 3. 资源管理 (Assets Manager)

在左侧边栏点击 **Assets**，进入资源管理界面。

- **存放方式**：所有要在游戏中使用的图片和音乐，请**直接复制**放入您电脑项目文件夹下的对应目录中：
  - `public/game/backgrounds/`（用于存放背景大图）
  - `public/game/characters/`（用于存放人物立绘透明 PNG）
  - `public/game/audio/`（用于存放 BGM 和音效，支持 mp3/wav/ogg）
- **编辑器显示**：复制完文件后，编辑器会自动侦测并在“背景图”、“角色图”、“音频”三个标签页下展示这些素材的缩略图或播放器。

> **💡 贴士**：拖拽上传功能正在开发中，目前请通过桌面文件管理器直接将图片放入对应文件夹。

---

## 🎭 4. 角色与变量 (Characters)

在左侧边栏点击 **Characters**，在这个面板您可以配置出场人物，无需记住他们复杂的英文 ID。

### 新建角色
1. 在“角色列表”顶部点击 **`+`** 号。
2. 弹窗中输入该角色的**底层 ID**（必须是纯英文，例如 `sakura`）。

### 配置角色
选中列表中的角色后，在右侧面板修改它的属性：
- **Display Name (显示名字)**：游戏中对话框显示的名字（如：木之本樱）。
- **Name Color (名字颜色)**：点击颜色拾取器，为该角色指定一个专属的名字弹幕颜色。
- **Expressions (表情差分)**：点击 `Add Expression` 可以为该角色添加无限张表情图。
  - 左侧格子填写表情名（如 `smile`, `angry`, `normal`）。
  - 右侧格子填写资源路径（如 `characters/sakura_smile.png`）。

*记得点击顶部的 **「Save Changes」** 保存您的配置。*

---

## 🎬 5. 编排剧情场景 (Scenes)

这是编辑器最核心的部分。在左侧边栏点击 **Scenes** 进入剧情编排界面。

### 5.1 创建并选择场景
游戏是由一个一个独立的“场景（Scene）”串联起来的（比如第一天、小巷战、大结局等）。
- 点击左侧的 **`+`** 创建一个新场景，并为其命名（底层 ID 需是纯英文）。
- 点击列表中的某个场景，中间的工作区就会切换至该场景的**指令时间轴**。

### 5.2 操作指令时间轴 (Timeline)
中间深色的每一行代表剧本执行的一条指令。引擎会**从上到下依次执行**这些指令。

- **添加指令**：在顶部下拉菜单中选择您要的操作（如：Dialogue(对话)、Play BGM(放音乐)、Show Character(亮出立绘)等），然后点击“Add Command”。
- **上下移动**：鼠标悬停在某条指令上，点击右侧出现的 `↑` 或 `↓` 按钮来调整它的播放顺序。
- **删除指令**：点击红色的 `✕` 按钮移除不需要的操作。

### 5.3 在 Inspector 中编辑指令参数
在中间的 Timeline 中**点击高亮**某一条具体的指令块，右侧的 **Inspector** 面板会立刻变成填空表单：

- 如果点的是 **Dialogue (对话)**：右侧会让您输入说话人（可留空作为旁白）以及具体的对白文案。
- 如果点的是 **Show Character (显示立绘)**：右侧会让您选择是哪个角色、用哪个表情（对应角色配置里写的表情名），以及站在左边、中间还是右边。
- 如果点的是 **Choice Menu (分支选项)**：您可以无限添加按钮选项，每个按钮都可以配置其显示的文字，以及点击后**要跳转到的目标场景 ID (`Jump scene`)**。

---

## 🕹️ 6. 运行测试游戏 (Play Preview)

在编排完剧情后，点击中间顶部的 **「Save Script」** 将数据写入本地。

随时点击左侧边栏最下方的绿色的 **「▶ Play Preview」** 按钮。
此时，电脑桌面会自动弹出一个崭新的全屏窗口（就是真实玩家拿到的发行版面貌），在里面您可以像普通玩家一样点击推进剧情，立刻体验您刚才在编辑器里做的修改！
